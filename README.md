# Food
Information about food ingredients and full recipes. Contains APIs to do tasks like
get the amount of calories in basic ingredients, or get the calories per serving of
a recipe.

# Development

Install the latest version of Ruby 3 and Rails 8 using the instructions at
https://guides.rubyonrails.org/install_ruby_on_rails.html

## Running Locally

Create the local SQLite database by running `bin/rails db:create`.

Start the server with `bin/rails server`.
